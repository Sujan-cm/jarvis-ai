# Jarvis AI

A local open-source AI assistant for Windows.

## Features

- PySide6 desktop interface
- Ollama local LLM
- Llama 3.3 support
- SQLite memory
- Modular plugin architecture

## Requirements

- Windows 10/11
- Python 3.11+
- NVIDIA GPU recommended
